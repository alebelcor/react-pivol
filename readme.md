# React PiVol

> "Volume control" meme in React/Hooks/Emotion.

## Run locally

* Clone repository
* `npm install`
* `npm start`

## Usage

Increase your volume by correctly entering the number <abbr title="Pi">π</abbr>.

Note: Press <kbd>ESC</kbd> to clear.
